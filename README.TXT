
PROJECT TITLE: Chess Game Simulation
PURPOSE OF PROJECT: This project simulates a complete chess game.
It aims to provide a platform for two players to play chess, adhering
to official chess rules, including special conditions like check, checkmate,
and pawn promotion.

VERSION or DATE: Version 1.0
HOW TO START THIS PROJECT:
    1. Ensure you have a compatible Java development environment installed (e.g., JDK 17 or later).
    2. Clone the project repository or download the source code.
    3. Open the project in your preferred IDE (e.g., NetBeans, IntelliJ IDEA).
